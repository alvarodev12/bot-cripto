# 🤖 Crypto Trading Bot (Binance API)

Este é um bot de trading automatizado para a Binance, desenvolvido em Node.js. Ele utiliza a média móvel simples (SMA) para tomar decisões de compra e venda de ativos (como BTC/USDT).

## 🚀 Funcionalidades

- Monitoramento de preço em tempo real via API da Binance.
- Cálculo automático de indicadores (SMA).
- Execução automática de ordens de mercado (Market Orders).
- Configuração de estratégias de entrada e saída.

## 🛠️ Tecnologias Utilizadas

- **Node.js**: Ambiente de execução.
- **Axios**: Para requisições HTTP à API da Binance.
- **Crypto-js**: Para autenticação e assinaturas de segurança.
- **Dotenv**: Para gerenciamento de variáveis de ambiente (segurança).

## 📋 Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:
- [Node.js](https://nodejs.org/)
- Uma conta na Binance (com chaves de API geradas).
